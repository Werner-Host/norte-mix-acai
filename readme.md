# tecnologias e fluxo
- o sistema deve ser contruido em react, em formato compativel com dispositivos mobile.
- o visual deve ser cheio de cores armonizadas  aos produtos oferecidos
- o sistema seve bem fluido e dinamico, para facilitar o uso e agilidade para o cliente concluir o pedido
- Primeiro atendimento será via whatsapp, que irá encaminhar o link de acesso ao "cardágio digital".
- no cardápio quero que seja um formulário dinamico e intuitivo para clicar em mais ou menos para adicionar e remover itens
- no rodapé tenha o total de produtos, a soma dos produtos adicionados e o custo de entrega depois da seleção do bairro da entrega
- antes de enviar o pedido é obrigatório a seleção do bairro da entrega ou marcar "retirada no local"
- o botão de enviar pedido fica desativado até o cliente selecionar um item e a opção de entrega
